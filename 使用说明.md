# 民宿管理系统 - 使用说明

## 一、怎么启动

### 1. 环境要求
- 电脑上装了 Node.js（建议 16 版本以上）

### 2. 启动步骤
1. 打开命令行（cmd 或 PowerShell），进入项目目录 `d:\mf-07`
2. 第一次使用先装依赖：
   ```
   npm install
   ```
3. 启动服务：
   ```
   node server.js
   ```
4. 打开浏览器，访问：`http://localhost:6000`

看到"民宿管理系统"的页面就说明成功了。

## 二、功能说明

系统有 5 个页面，顶部标签切换：

### 📅 日历房态（首页）
- 一目了然看到每套房每天的状态
- 蓝色条 = 已被预订，橙色 = 清洁日
- 可以切换查看某一套房
- 点击任意日期格子可以快速下单
- 上下月按钮切换月份

### 📋 订单管理
- 查看所有订单，可按状态筛选
- 每个订单可以查看详情
- 订单状态流程：待确认 → 已确认 → 入住中 → 已退房
  - 也可以取消订单（按退订规则扣款）

### 🏠 房源管理
- 添加、编辑、删除房源
- 每套房可设置：
  - 名称、可住人数、几室、设施描述
  - 入住/退房时间
  - 平日价格、周末价格、节假日价格

### 🧹 清洁排期
- 下单成功后，退房当天自动生成一条清洁任务
- 可以按日期筛选
- 清洁完成后点"完成"按钮标记

### 📊 统计报表
- 按月查看每套房的：
  - 入住晚数
  - 入住率（百分比）
  - 收入金额
- 顶部有总览：总收入、总入住晚数、平均入住率

## 三、核心规则说明

### 1. 撞单检测（最最重要！）
- 同一套房，同一天只能有一个订单
- 下单时系统自动检查，撞单会直接报错不让下单
- 退房当天可以接新订单（比如 A 客人 15 号退房，B 客人 15 号入住，这是允许的）
- 取消的订单不算，日期会自动释放出来

### 2. 价格计算
- 按晚算钱，住几晚算几晚
- 周一到周五算平日价，周六周日算周末价
- 节假日价格暂时没用上，可后续扩展

### 3. 退订规则
- 提前 7 天以上取消：全额退款
- 提前 3-7 天取消：扣 50% 违约金
- 提前 1-3 天取消：扣 80% 违约金
- 当天或入住后取消：不退钱

### 4. 清洁安排
- 每个订单确认后，自动在退房那天排一条清洁
- 取消订单时，对应的清洁也会自动取消
- 清洁完成记得标记一下

## 四、快速上手操作流程

### 第一步：登记房源
1. 点顶部"🏠 房源管理"
2. 点右上角"+ 添加房源"
3. 填名字、价格等信息，保存
4. 有几套房就加几套

### 第二步：试一下下单
1. 切回"📅 日历房态"
2. 点一个空着的日期格子
3. 选退房日期，填客人名字，点确认下单
4. 看看日历上是不是出现了蓝色条

### 第三步：试一下会不会撞单
1. 再点同一个日期，试试再下一单
2. 系统应该会报错说日期冲突
3. 这就对了，撞单被挡住了

### 第四步：试一下退订
1. 点"📋 订单管理"，找到刚才的订单
2. 点"详情"，再点"取消订单"
3. 确认后看看违约金多少
4. 回到日历，日期是不是又空了

### 第五步：看看清洁排期
1. 先再下一个订单
2. 点"🧹 清洁排期"
3. 看看退房那天是不是自动有清洁任务了

## 五、文件清单

项目里有这些文件：

| 文件名 | 作用 |
|--------|------|
| `server.js` | 主程序，后端 API 都在这里 |
| `db.js` | 数据库相关，日期工具、撞单检测核心逻辑 |
| `test.js` | 测试文件，35 个测试用例 |
| `package.json` | 项目配置和依赖说明 |
| `public/index.html` | 前端页面（一个文件包含所有） |
| `data.db` | 数据库文件（第一次运行后自动生成） |
| `使用说明.md` | 就是这个文件 |

## 六、怎么跑测试

如果想验证系统靠不靠谱，可以跑测试：

```
node test.js
```

会跑 35 个测试用例，重点测：
- 日期计算（跨月、跨年、闰年）
- 撞单检测（各种重叠情况）
- 价格计算（工作日、周末）
- 退订规则（不同提前天数）
- 跨月跨年边界情况

全部通过说明核心逻辑没问题。

## 七、常见问题

**Q: 关闭命令行窗口服务就停了？**
A: 是的，这个是小程序，关了就停。要用的时候开着就行。

**Q: 数据存在哪？**
A: 都在 `data.db` 这个文件里，备份这个文件就等于备份全部数据。

**Q: 节假日价格怎么用？**
A: 目前没实现自动识别节假日，如果需要可以后续加，或者直接手动改价格。

**Q: 清洁阿姨时间撞了怎么办？**
A: 目前只按天排清洁，同一天可以有多个清洁任务（不同的房）。如果阿姨人手不够，需要你自己协调安排，系统只管记录。
